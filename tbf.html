Turn-Based Football

Download Here

This game was part of my Final Major Project at University, where I created a football simulation game with a turn based strategy spin. I decided to pursue this idea because I wanted to improve my ability to create AI. This was a solo project with a fast turn around, which I completed within an eight week window. This game was made in UE4 using Blueprint and C++, and required me to work in UI, AI, animation and tool programming.
